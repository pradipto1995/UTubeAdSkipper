# UTubeAdSkipper Chrome Extension

## About This Project
Simple Chrome Extension to skip youtube Ads.

## Getting Started
### Steps:-
#### 1.Download and extract the zip file.
![image](https://user-images.githubusercontent.com/55427745/219848587-cc27f89b-70bc-4a6f-845e-574c8c8b8ef5.png)

#### 2.Open up google chrome and go to chrome://extensions/ 
![image](https://user-images.githubusercontent.com/55427745/219848619-265f2a28-bd80-47b5-ad11-41af75a53b4c.png)


#### 3.Turn on Developer mode
![image](https://user-images.githubusercontent.com/55427745/219848630-e8971b2a-4f1d-42c3-934a-c43e68d98369.png)


#### 4.Select load unpacked
![image](https://user-images.githubusercontent.com/55427745/219848669-bb76652c-b94b-4ed0-9731-9495113fc3a1.png)


#### 5.Browse and select the extracted folder
![image](https://user-images.githubusercontent.com/55427745/219848760-e62a1552-7e40-46c2-9483-c29edc314733.png)

#### 6.Enjoy!!
![image](https://user-images.githubusercontent.com/55427745/219848783-6b9cd8ca-9321-4101-96da-9415a55ec5e5.png)
